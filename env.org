* Editor

Use [[file:nvim.org][Neovim]] as default editor

#+BEGIN_SRC sh
ALTERNATE_EDITOR=""
EDITOR=nvim
VISUAL=nvim
#+END_SRC

* COMMENT Pager

Use bat (cat clone) as pager

#+BEGIN_SRC sh
PAGER=bat
#+END_SRC

* Input method

Use [[file:fcitx.org][fcitx]] ad input method

#+BEGIN_SRC sh
GTK_IM_MODULE=fcitx
QT_IM_MODULE=fcitx
GLFW_IM_MODULE=ibus
XMODIFIERS=@im=fcitx
#+END_SRC

* Firefox Wayland

Force Firefox to run using Wayland on Wayland.

#+BEGIN_SRC sh
MOZ_ENABLE_WAYLAND=1
#+END_SRC
