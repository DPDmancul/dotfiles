# i3 (window manager)

Unfortunately nvidia does not yet support wayland, so I need to fallback on i3 for the moment, when using nvidia.

```nix modules/system/i3.nix
{ config, pkgs, lib, ... }:
{
  services.xserver = {
    enable = true;

    desktopManager.xterm.enable = false;

    windowManager.i3 = {
      enable = true;
      package = pkgs.unstable.i3; # TODO stable after gaps being merged
      extraPackages = [ ];
    };
  };
}
```
