# Internationalization

Use the default locale (POSIX C) for all except paper size:

```nix "config" +=
i18n = {
  defaultLocale = "C.utf8";
  extraLocaleSettings = {
    LC_PAPER = "it_IT.utf8";
  };
};
```

## Input method

Use fcitx5 as input method

```nix "config" +=
i18n.inputMethod.enabled = "fcitx5";
```
