Inspired by [[https://writepermission.com/introducing-literate-dotfiles.html][Toon Claes]], these dotfiles are managed by [[https://orgmode.org][OrgMode]] literate programming.

View [[https://dpdmancul.gitlab.io/dotfiles][HTML version]].

* Usage

1. Clone the repo [[https://gitlab.com/DPDmancul/dotfiles.git]]
2. Just run ~make~ to update all dotfiles; ~make publish~ will generate the HTML for these dotfiles.

*Warning* read the dotfiles before apply them!
