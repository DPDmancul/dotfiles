My dotfiles for NixOs and the Nix package manager, written in literate
programming using Org mode.

View [[https://dpdmancul.gitlab.io/dotfiles][HTML version]].

Download [[https://gitlab.com/DPDmancul/dotfiles/-/jobs/artifacts/nix/download?job=tangle][tangled dotfiles]].

* Usage

1. Clone the repo [[https://gitlab.com/DPDmancul/dotfiles.git]]
2. Optionally run ~nix-shell~ to get all the required dependencies into the
   environment
3. Just run ~make~ to update all dotfiles;
   ~make doc~ will generate the HTML for these dotfiles.

*Warning* read the dotfiles before apply them!

** Without Emacs

Download [[https://gitlab.com/DPDmancul/dotfiles/-/jobs/artifacts/nix/download?job=tangle][tangled dotfiles]] and copy them.
